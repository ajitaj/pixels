International Animation Certificate

Duration : 6 Months
Description
This workshop is focused on introducing the principles of character animation, body mechanics, the planning process, industry fundamentals and insights that are used in developing a memorable performance.

Benefits
Students will learn how to manage and animate while reinforcing and applying the knowledge learned in order to create convincing and appealing animations. This course sets the groundwork and foundation for advanced learning in Character Animation.

Assignments
TBD, Pending the rigs and character used.

Breakdown Level 1

+  Principles of Animation Part 1
+ Principles of Animation Part 2
+ Weight, Spacing and Timing (Balloon, Bowling, Tennis)
+ Reactive and Interactive (Dynamic pinball, Ball maze)
+ What and How to create a Character (Thinking ball, Jumping squirrel ball)
+ Pose vs Gesture
+ Silhoutte, Shapes, Contrast
+ Squash and stretch, Up and Downs
+ Weight on a character (Get up from chair)
+ Shifting weight ona character (reach, turn on spot)
+ Moving weight on character (jumping forward)

Breakdown Level 2

+  Animation Theory-Communication (Posing, Story, Appeal)
+ Animation Planning and Process (Gathering & Editing of Reference)
+ Getting and Using the Best References (Application of reference)
+ Balance and Arcs on a Character (Applying Reference to CG Character)
+ Animation Theory - Advanced Timing (Applying & controlling timing)
+ What is a Character? Personality? Behaviors? (Theory)
+ What is Blocking
+ Blocking Plus, Second Pass
+ Spine Pass
+ Polish Pass
+ Techniques of Animations (P2P, Layered, Straight Ahead)
